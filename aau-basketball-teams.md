## Data Structure Reference

Each tryout entry in `tryouts.json` includes:
- `id`: Unique identifier (kebab-case)
- `teamName`: Team display name
- `organizationName`: Parent organization
- `ageGroup`: 14U, 15U, 16U, 17U, or "High School"
- `gradeLevel`: Grade range (e.g., "9th-10th")
- `gender`: Boys, Girls, or Co-ed
- `tryoutDate`: ISO date format (YYYY-MM-DD)
- `tryoutEndDate`: (optional) For multi-day tryouts
- `startTime`: Tryout start time
- `endTime`: (optional) Tryout end time
- `venue`: Facility name
- `address`: Street address
- `city`: City name
- `zipCode`: (optional) Postal code
- `contactEmail`: (optional) Contact email
- `contactPhone`: (optional) Contact phone
- `websiteUrl`: Team/organization website
- `registrationUrl`: (optional) Registration link
- `registrationDeadline`: (optional) ISO date format
- `cost`: (optional) Fee information
- `notes`: Additional details
- `skillLevel`: All Levels, Competitive, or Advanced
- `createdAt`: ISO timestamp
- `updatedAt`: ISO timestamp
- `source`: Data source (e.g., "Team Website")
